# 🐼 Goods Panda

> *A panda walked into a store. He bought some shoes, a bag, and a TV. Then he left. No one questioned it. This is his store now.*

---

## 🤔 What is this?

It's a **responsive e-commerce landing page** for a panda who somehow got into retail. Built with just **HTML & CSS** — no frameworks, no npm install hell, no 47 dependency vulnerabilities. Just vibes and divs.

---

## 🧱 Tech Stack

- **HTML5** — the bones 💀
- **CSS3** — the drip 💅
- **Google Fonts** — Bebas Neue + Nunito (fancy panda hours)
- **JavaScript** — 0 lines. Zero. We don't do that here. 🚫
